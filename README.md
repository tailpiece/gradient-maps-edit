# Gradient Map Editor for ComicShare

ComicShareで自動着色した時の表示を、疑似確認できる


## 状況

- サンプル画像データはコミpo
- ドラッグドロップで色変更可
- ドラッグドロップで画像登録可
- レスポンシブ対応

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build
```

Running

```sh
npm run dev
```
