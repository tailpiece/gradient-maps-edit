# Gradient Map Editor for ComicShare

ComicShareで自動着色した時の表示を、疑似確認  
https://tailpiece.github.io/gradient-maps-edit/

## Building and running on localhost

First install dependencies:

```sh
yarn install
```

To create a production build:

```sh
yarn run build
```

Running

```sh
yarn run dev
```
